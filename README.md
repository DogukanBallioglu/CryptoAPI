# 🚀 Crypto Tracker - API Learning Project

Bu proje, modern web geliştirmenin temeli olan **API (Application Programming Interface)** kullanımı, **Asenkron JavaScript (Async/Await)** ve **Dinamik DOM Yönetimi** konularını pekiştirmek amacıyla geliştirilmiştir.



## 🎯 Projenin Amacı
Bu uygulamayı geliştirirken temel odak noktam, bir sunucudan canlı veri çekme sürecini (Data Fetching) uçtan uca deneyimlemekti. Proje kapsamında şunları uyguladım:
* **Fetch API** kullanarak gerçek zamanlı finansal verilere ulaşmak.
* `async/await` yapısı ile asenkron veri akışını yönetmek.
* Gelen JSON verisini parse edip HTML içerisine dinamik olarak yerleştirmek.
* Hata yönetimi (`try/catch`) ile ağ veya API kaynaklı hataları kontrol altına almak.

## 🛠️ Kullanılan Teknolojiler
* **HTML5 & CSS3:** Glassmorphism (Cam efekti) ve responsive tasarım.
* **JavaScript (ES6+):** Dinamik içerik yönetimi ve asenkron programlama.
* **CoinGecko API:** Gerçek zamanlı kripto para verileri için kullanılan REST API.

## 💡 Neler Öğrendim?
1.  **API Entegrasyonu:** İstemci (Client) ve Sunucu (Server) arasındaki veri alışverişi nasıl gerçekleşir?
2.  **JSON Veri Yapısı:** API'den gelen ham veri paketleri içinden spesifik bilgilere (`current_price`, `image` vb.) nasıl erişilir?
3.  **DOM Manipülasyonu:** JavaScript ile HTML elementlerini (div, img, span) kod tarafında oluşturup arayüze nasıl enjekte edilir?
4.  **Modern UI:** `backdrop-filter`, `flexbox` ve `linear-gradient` gibi CSS özellikleriyle kullanıcı deneyimi nasıl artırılır?

## 🚀 Nasıl Çalıştırılır?
1.  Bu repoyu bilgisayarınıza clone'layın veya ZIP olarak indirin.
2.  `index.html` dosyasını herhangi bir modern tarayıcıda açın.
3.  "Fiyatları Güncelle" butonuna basarak canlı verileri anlık olarak çekin!
